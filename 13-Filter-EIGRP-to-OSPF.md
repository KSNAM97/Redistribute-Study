# 13. EIGRP → OSPF Filtering (Metric-Type / Tag)

> Route-map을 사용한 Metric Type 및 Tag 부여

---

## EX4) EIGRP → OSPF 조건별 재분배

### 📋 요구사항

| 조건 | 처리 |
|------|------|
| `4.1.1.0/24`, `198.198.x.0/24` | Metric 고정 200 (E2) |
| 나머지 EIGRP 네트워크 | 누적 Metric (E1), **TAG 300** |

---

### 🔧 R3

```bash
ip prefix-list EIGRP_E2 permit 4.1.1.0/24
ip prefix-list EIGRP_E2 permit 198.198.0.0/16 le 32
!
route-map EIGRP-OSPF_REDI permit 10
 match ip address prefix-list EIGRP_E2
 set metric-type type-2
 set metric 200
route-map EIGRP-OSPF_REDI permit 20
 set metric-type type-1
 set tag 300
!
router ospf 1
 redistribute eigrp 100 metric-type 1 subnets route-map EIGRP-OSPF_REDI
```
