# 06. RIPv2 → OSPF Redistribute

> R1에서 RIPv2 네트워크 정보를 OSPF 환경으로 재분배

---

## EX5) RIPv2 → OSPF

### 📋 요구사항

- RIPv2가 OSPF 환경으로 재분배 시 Metric 값이 고정되어야 함 (E2 기본)
- RIPv2에 포함된 Connected 네트워크도 함께 재분배됨

### 🔧 R1

```bash
router ospf 1
 redistribute rip subnets
```
