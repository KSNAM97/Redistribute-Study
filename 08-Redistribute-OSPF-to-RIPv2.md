# 08. OSPF → RIPv2 Redistribute

> 서로 다른 Routing Protocol 간 재분배 시 두 가지 Protocol이 enable 된 Router에서 설정

---

## EX) OSPF → RIPv2

### 📋 요구사항

- R4에서 OSPF로부터 재분배된 모든 네트워크 정보의 Metric = **3** 으로 확인

### 🔧 R1

```bash
router rip
 redistribute ospf 1 metric 3
```
